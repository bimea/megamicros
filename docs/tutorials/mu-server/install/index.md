# Installing the Megamicros broadcast server
In this tutorial, we propose several installation methods, depending on your system.
These installations are intended for developers. 
If you only wish to use *megamicros-server*, download the *megamicros* binaries instead.

The last section is about running a server behind a 4G router.

* [Installing on MacOS system](mac.md)
* [Installing on an industrial PC](minipc.md)
* [Installing on a jetson nano card](jetson.md)
* [Installing on a Raspberry Pi](raspi.md)
* [Installing on a Ornge Pi 4G PC](orangepi3b.md)
* [Installing behind a 4G router](4Grouter.md)
