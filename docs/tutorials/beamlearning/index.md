# Machine learning for beamforming

The *Megamigros* python library provides two applications and somme additional tools for writing AI applications that embed one or more microphones array.

* [**AiDB**](aidb.md) is the REST database application for audio signals storing and labeling;
* [**AiBoard**](aiboard.md) is a frontend for the **AiDB** REST database server;
* [**AiLab**](ailab.md) provides tools and applications examples as notebooks for AI algorithms writing.
* [**Beamforming**](beamforming/index.md)
